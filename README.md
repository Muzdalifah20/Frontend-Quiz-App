# Quiz App

A simple quiz app made with HTML, CSS, and JavaScript.

## What it does

- Choose a quiz topic.
- Answer the questions.
- See your score at the end.
- Switch between light and dark theme.

## Features

- Keyboard support.
- Focus styles for accessibility.
- Theme saved in local storage.
- Quiz data loaded from JSON.

## How to run

1. Open the project folder.
2. Open `index.html` in your browser.

# Live Site

https://muzdalifah20.github.io/Frontend-Quiz-App/starter-code/

## Files

- `index.html` — page structure.
- `main.css` — styles.
- `main.js` — app logic.
- `assets/` — images, icons, fonts, and quiz data.

## Notes

- This app uses custom radio buttons.
- It works with keyboard and mouse.
